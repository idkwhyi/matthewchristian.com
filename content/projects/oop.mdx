# Math Master

A math educational game created using the java programming language to fulfill my final group assignment in the object oriented programming course.

This project has several main features such as:
- Login and Register user account
- Level and level difficulty
- Questions for each level
- Rewards in the form of coins that can be used to unlock levels

This project is also connected to the MySQL database as its main storage.
